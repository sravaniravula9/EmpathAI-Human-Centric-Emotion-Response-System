🤖 EmpathAI - Human-Centric Emotion Response System:

EmpathAI is an AI-powered web application that detects human emotions through real-time facial expression analysis and generates empathetic responses. The system uses computer vision and machine learning to understand users' emotional states and provide supportive, human-like interactions.

---
📌 Features

- 😊 Real-time facial emotion detection
- 🎥 Webcam-based emotion recognition
- 🧠 AI-generated empathetic responses
- 💬 User-friendly web interface
- ⚡ Fast and accurate emotion prediction
- 📊 Supports multiple emotions such as:
  - Happy
  - Sad
  - Angry
  - Fear
  - Surprise
  - Neutral

🚀 How to Run
1. Clone the repository
```bash
git clone https://github.com/sravaniravula9/EmpathAI-Human-Centric-Emotion-Response-System.git
```
2. Navigate to the project folder
```bash
cd EmpathAI-Human-Centric-Emotion-Response-System
```
3. Install dependencies
```bash
pip install -r requirements.txt
```
4. Run the application
```bash
python app.py
```
5. Open your browser and visit
```
http://127.0.0.1:5000
